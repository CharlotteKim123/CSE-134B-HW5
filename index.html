Projects

Emotion Detector

"Emotion Detector": This website allows one to see the overall review of a product (i.e. Amazon product). All you have to do is input the link to the product into the given text box. The website will then return the product's overall word cloud, positive word cloud, negative word cloud, a bar graph and a circle graph showing the positive vs negative reviews, and the overall review of the product.
The Game of Life

"Game of Life": This program creates a new board by reading a file and setting the rows and columns of the board on the file, where the first line is the number of rows, and the second line being the number of columns. We also simulate the "steps" generations for each board.

 Cipher

"Cipher": This program uses a cipher to encrypt the inputted files.
